# Zephyr RTOS Project

This project demonstrates RTOS concepts using Zephyr RTOS on STM32.

## Board
STM32 Nucleo H7A3ZI-Q

## Features (planned)
- Multithreading
- Mutex (resource management)
- Priority inversion demonstration
- Hardware interaction (LED)

## Build
west build -b nucleo_h7a3zi_q <project_path>
